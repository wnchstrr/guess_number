# ./guess_number.py — can you beat it in under 7 tries?

> A number guessing game written in Python. No dependencies, no setup — just run
> it.

---

## Quick start

```bash
git clone https://github.com/wnchstrr/guess_number.git
cd guess_number
python main.py
```

---

## How it works

The computer picks a number between 1 and 100. You guess. It tells you higher or
lower. First to zero in wins — your attempt count is shown at the end.
